Roadnet® Transportation Suite Distribution Optimization and Control Complex

Developed by UPS Logistics Technologies based on detailed Transnavicom maps, it allows solving daily distribution issues: planning optimal product delivery routes, optimizing sales and sales agent routes, tracking, monitoring and analyzing distribution processes in real time. The complex contains a complete package of software products that closely interact with each other, allowing you to solve the tasks as efficiently as possible.

Electronic cards (Professional version)

Electronic maps (Professional version) will allow you to add your own layer of objects, logos, text labels and comments to the map and automatically link them to pre-set addresses with display on the map. PURPOSE: Maintaining a database with display of the location of objects on the city map for the purpose of visual analysis of the situation.

Electronic Maps (Enterprise Edition)

Electronic maps (Enterprise Edition) is an information and reference guide to major cities in Ukraine and Russia, combining detailed electronic maps of terrain, roads, public transport routes and enterprise databases. Electronic maps (Enterprise Edition): combine all maps of the Base version series on one DVD; provide quick transition from one map to another without rebooting the software shell; work without being tied to a disk and over a local network. ATTENTION: The program package includes a network HASP key for 10 workstations.

Electronic maps (Net Edition)

Electronic Maps (Net Edition) is a special software package that combines all the features of Electronic Maps (Base version) with the undeniable advantages of working over a network and without being tied to a disk. The main distinguishing feature of the Electronic Maps (Net Edition) product is a unique designer, with which you can equip the software shell with the maps you need yourself, without overpaying for unnecessary ones. Electronic Maps (Net Edition): provide a unique designer that allows you to combine only the Base version series maps you need on one DVD; provide a quick transition from one map to another without rebooting the software shell; work without being tied to a disk and over a local network. ATTENTION: The program package includes a network HASP key for 5 workstations.

Taxi Coordinator

Taxi-Coordinator is a new generation computer system designed for fast and convenient organization of taxi dispatch services.
